# Wireframes for Macellum Media Website

This document outlines the wireframes for the key sections of the Macellum Media website, based on the sitemap and inspired by Hyperflow.co.

**General Notes:**
*   These are low-fidelity wireframes focusing on layout, content hierarchy, and user flow.
*   Specific styling (colors, fonts, final imagery) will be defined in the mockup phase.
*   "[L]" denotes the Macellum Media Logo.
*   "[NAV]" denotes the main navigation menu.
*   "[CTA]" denotes a Call to Action button.
*   "[IMG]" denotes a placeholder for an image or visual element.
*   "[ICON]" denotes a placeholder for an icon.
